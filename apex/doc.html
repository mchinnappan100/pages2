Apex Analyzer Documentation

A tool to analyze Salesforce Apex classes and generate inline documentation or test classes.

Overview

Apex Analyzer is a web application designed for Salesforce developers. It allows you to fetch Apex classes from raw GitHub URLs or paste code directly, then generates inline documentation or test classes using a backend server connected to a local Gemma LLM. The app features a clean, dark-themed interface for a seamless user experience.
