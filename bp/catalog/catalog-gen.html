Command Examples

Generate UUIDs locally:

npm install -g uuid-gen-cli
uuid-gen -n 10 > uuids.txt

CSV format example:

Id,External_Id__c
701xxx,6d0f8c7f-8f8a-4d6a...
701yyy,53ce9a12-4a6f-44a8...

Bulk upsert command:

npm install -g sf-bulk-upsert-uuid
sf-bulk-upsert-uuid --object Product2 \
--file uuids.csv --field External_Id__c
