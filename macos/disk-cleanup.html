macOS Disk Cleanup Guide

Free 20–50 GB Space Safely on macOS

Learn how to clean npm cache, developer files, system caches, simulators, and hidden folders safely.

Overview

macOS stores temporary files to improve performance. Over time these accumulate and consume significant disk space.

  • .cache folder
  • .npm cache
  • Library/Caches
  • Xcode Developer files
  • Simulator files

Cleaning ~/.cache

This folder stores cache from developer tools and browsers. 


du -sh ~/.cache
rm -rf ~/.cache/*

Safe and frees 5–20 GB

Cleaning npm Cache


npm cache clean --force

npm cache can grow very large.

Cleaning User Cache


rm -rf ~/Library/Caches/*

Safe. Cache will regenerate automatically.

Cleaning System Cache


sudo rm -rf /Library/Caches/*

Some protected folders cannot be deleted. This is normal.

Cleaning Xcode Files


rm -rf ~/Library/Developer/Xcode/DerivedData
rm -rf ~/Library/Developer/Xcode/Archives

Frees 2–20 GB.

Cleaning Simulator Files


rm -rf ~/Library/Developer/CoreSimulator/Devices

Frees several GB safely.

Full Cleanup Script 


npm cache clean --force
rm -rf ~/.cache/*
rm -rf ~/Library/Caches/*
rm -rf ~/Library/Developer/Xcode/DerivedData
rm -rf ~/Library/Developer/CoreSimulator/Devices

Check Disk Usage


df -h
du -sh ~/*
