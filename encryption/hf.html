Hyperforce Status Guide

How to Verify Hyperforce

Hyperforce is Salesforce’s next-generation infrastructure architecture built on public cloud (AWS). Use these methods to determine if your organization has been migrated.

1

Check Instance Name in Setup

  1. • Navigate to Setup.
  2. • In Quick Find, type Company Information.
  3. • Locate the Instance field.

Hyperforce

5-character alphanumeric ending in "S".

e.g., USA2S, IND1S, DEU3S

Legacy (1st Party)

2-letters + numbers pattern.

e.g., NA142, EU44, AP25

2

Use Salesforce Trust

Verify status by searching your instance ID at trust.salesforce.com.

Trust Tip

If the instance is Hyperforce, the description will explicitly mention the region, e.g., "Hyperforce - United States".

3

Command Line Interface (CLI)

Run the following command to see instance metadata:

sf org display --target-org <alias_or_username>
