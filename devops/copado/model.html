Copado Data Model

User Story

Represents a unit of work. Typically linked to one feature branch.

Feature Branch

Contains one or more commits associated with a user story.

Promotion

Deployment event that includes one or more user stories.

Relationships

  • User Story → 1 Feature Branch (recommended)
  • Feature Branch → Many Commits
  • Promotion → Many User Stories
  • Promotion → Indirectly deploys commits
