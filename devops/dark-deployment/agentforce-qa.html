Winter '26 Terminology Note
  • Topics → SubAgents: Each routing category is now a first-class SubAgent with its own scope and instructions.
  • Top-level Agent → Orchestrator Agent: The parent that delegates to SubAgents.
  • Topic Instructions → Job To Be Done: Instructions within a SubAgent are framed as a Job To Be Done.
  • Agent-to-Agent → Agent Network: Multi-agent coordination is now called an Agent Network.
  • Questions use both old and new terms — explanations clarify the mapping where relevant.
