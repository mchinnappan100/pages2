Salesforce DKIM Complete Setup Guide

Domain Configuration

What is DKIM?

DKIM (DomainKeys Identified Mail) allows Salesforce to digitally sign emails using your domain. This improves trust, prevents spoofing, and improves email deliverability.

Emails will be signed as: example.com instead of salesforce.com.

Step-by-Step Salesforce Setup

  1. Login to Salesforce
  2. Go to Setup
  3. Search for DKIM Keys
  4. Click Create New Key
  5. Enter values:
    • Selector: selector1
    • Alternate Selector: selector2
    • Domain: example.com
    • Key Size: 2048
  6. Click Save
  7. Copy generated DNS records
  8. Add DNS records in DNS provider
  9. Wait for DNS propagation
  10. Click Activate in Salesforce
  11. Status should show Active
  12. Send test email and verify DKIM PASS
