NPM Proxy Setup Guide

📚 Overview

In corporate networks, internet traffic is routed through a proxy server. NPM must be configured properly to communicate with the registry.

🌐 Network Flow

graph LR A[Developer Machine] -->|npm install| B[Corporate Proxy] B --> C[Firewall] C --> D[NPM Registry] D --> C C --> B B --> A

⚠️ Common Errors


npm ERR! network getaddrinfo ENOTFOUND registry.npmjs.org
npm WARN invalid config proxy

💻 Setup Proxy


npm config set proxy http://username:password@proxy.company.com:8080
npm config set https-proxy http://username:password@proxy.company.com:8080

🧹 Remove Bad Config


npm config delete proxy
npm config delete https-proxy

✅ Verify


npm config list
npm ping

⚙️ Environment Variables


set HTTP_PROXY=http://username:password@proxy.company.com:8080
set HTTPS_PROXY=http://username:password@proxy.company.com:8080

🚀 Best Practices

  • Do not store credentials in .npmrc
  • Prefer environment variables
  • Test with curl before npm
