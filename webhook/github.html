GitHub WebHook Setup

  1. Go to Repository → Settings → Webhooks
  2. Payload URL: https://api.example.com/webhooks/github
  3. Content type: application/json
  4. Secret: shared HMAC secret
  5. Select events (push, pull_request, workflow_run)
  6. Save and test delivery

Headers to Validate 

X-GitHub-Event
X-Hub-Signature-256
X-GitHub-Delivery
