HMAC Signature Verification 
X-Hub-Signature-256: sha256=abc123...

computed = HMAC(secret, raw_body)
compare(computed, received)
  • Prevents payload tampering
  • Ensures sender authenticity
  • Always use constant-time comparison

Secrets Management

  • Store secrets in Vault / KMS / environment variables
  • Rotate secrets periodically
  • One secret per integration

IP Allowlisting

  • Restrict inbound traffic to known provider IP ranges
  • Update automatically (GitHub & Atlassian publish ranges)
