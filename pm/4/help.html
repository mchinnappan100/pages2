✨ Key Features

  • 📝 Task Management: Add, edit, delete, and track tasks with priorities, statuses, progress, dependencies, and assignees.
  • 📊 Gantt Chart: Visualize your project timeline and task durations interactively.
  • 👥 Resource Management: Manage people, equipment, materials, and budgets with availability and skills.
  • 📈 Reports & Analytics: Get instant project overviews, progress, status breakdowns, and resource utilization.
  • 📅 Project Calendar: See all tasks on a monthly calendar, with full task names and PDF export.
  • 💾 Import/Export: Save and load projects as JSON files. Drag-and-drop supported for easy import.
  • 🖨️ PDF Export: Print your project calendar as a PDF with project name as title.
  • 🔒 Auto-Save: Your last project is remembered in your browser (local storage).
  • 🌙 Modern UI: Clean, responsive design with emoji-enhanced navigation and sticky footer.
  • ❤️ Open Source & Free: No login, no subscription, no data leaves your device.
