What is a Vector Database?

Embeddings

Data is transformed into numerical vectors representing semantic meaning. Similar meanings are mathematically close in vector space.

Similarity Search

Search uses cosine similarity, Euclidean distance, or inner product instead of exact keyword matching.

AI Applications

Used in semantic search, recommendation systems, chat memory, retrieval augmented generation, anomaly detection, and clustering.

PostgreSQL Vector Support with pgvector

Native Vector Type

Store vectors directly inside PostgreSQL using the vector(n) type.

Approximate Nearest Neighbor

Supports HNSW and IVFFlat indexes for fast similarity search at scale.

SQL + AI Together

Combine traditional relational queries with vector search seamlessly. 

CREATE EXTENSION vector;

CREATE TABLE documents (
  id bigserial PRIMARY KEY,
  content text,
  embedding vector(1536)
);

CREATE INDEX ON documents
USING hnsw (embedding vector_cosine_ops);

SELECT content
FROM documents
ORDER BY embedding <=> '[...]'
LIMIT 5;
