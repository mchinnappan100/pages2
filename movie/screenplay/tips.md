**Key Features:**
- **Standard screenplay formatting** - Scene headings, action, character, dialogue, parenthetical, and transition elements
- **Auto-formatting** - Press Tab to cycle element types, Enter to create new formatted elements
- **Character tracking** - Automatically detects and lists all characters
- **Scene navigation** - Lists all scenes for quick reference
- **Quick insert buttons** - INT/EXT, transitions, and common screenplay elements
- **Real-time statistics** - Page count, scene count, character count, and estimated runtime
- **Script notes** - Dedicated area for production notes
- **Save/Load functionality** - Save scripts as JSON files and reload them later
- **PDF export** - Print-ready formatting for professional distribution
- **Dark theme** - Easy on the eyes for long writing sessions
- **Courier font** - Industry-standard screenplay typeface

The app follows professional screenplay formatting standards with proper margins and spacing, making it suitable for actual script production.
