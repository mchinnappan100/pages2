🔤

BPE Tokenizer

Byte-level base vocab (256) + 4 special tokens + 7932 learned merges. Priority-queue encoding O(n log n) — encodes 5.4M chars in ~3s. Word-level cache avoids re-encoding repeated tokens.

🔄

RoPE + KV Cache

Rotary Position Embeddings replace learned absolute positions. KV cache stores K and V tensors per layer. Position offset ensures token N gets angle θ·N not θ·0.

🎲

Sampling Stack

Temperature → Top-k → Top-p nucleus → Min-p adaptive filter → Repetition penalty (frequency-aware) → Presence penalty (flat) → Multinomial draw.

⚖️

Weight Tying

Token embedding matrix is shared with the LM head output projection. Saves 4M parameters and improves generalisation — gradient flows through both input and output paths.

⚡

Mixed Precision

bfloat16 inference on MPS / CUDA halves memory bandwidth. Training uses torch.autocast; optimiser states stay in float32 for numerical stability.

📐

SwiGLU + GQA

SwiGLU (Shazeer 2020) replaces GELU FFN with gated activation. GQA (2 KV heads vs 4 Q heads) cuts KV cache memory by 2× at inference time.
