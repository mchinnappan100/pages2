Architecture Overview

What we are building and how the five classes fit together

We build a classic Encoder–Decoder Transformer as introduced in "Attention is All You Need" (Vaswani et al., 2017). The encoder reads a source sequence and produces context vectors; the decoder generates a target sequence one token at a time, attending to both its own previous outputs and the encoder context.

The implementation is split into five classes that mirror the original paper's structure precisely:

Encoder–Decoder Transformer — Class Map Source tokens [src_seq_len, batch] ↓ nn.Embedding × √d_model PositionalEncoding ← sinusoidal pe added to embedding ↓ TransformerEncoder (stack of N layers) └── TransformerEncoderLayer × N Multi-Head Self-Attention + Add&Norm Feed-Forward Network + Add&Norm ↓ memory [src_seq_len, batch, d_model] ↓ ─────────────────────────────────────────────────────┐ Target tokens [tgt_seq_len, batch] │ ↓ nn.Embedding × √d_model │ PositionalEncoding │ ↓ │ TransformerDecoder (stack of M layers) │ └── TransformerDecoderLayer × M │ Masked Self-Attention + Add&Norm │ Cross-Attention (Q←tgt, K/V←memory) + Add&Norm ◄──────┘ Feed-Forward Network + Add&Norm ↓ nn.Linear → logits [tgt_seq_len, batch, tgt_vocab_size]

02

Positional Encoding

Injecting token-order information via sinusoidal functions

The Transformer has no recurrence or convolution, so it needs a way to know where each token sits in the sequence. Positional encoding adds a fixed, deterministic signal to the embedding of each token. The signal at position pos and dimension i is:

PE[pos, 2i] = sin(pos / 10000^(2i/d_model))
PE[pos, 2i+1] = cos(pos / 10000^(2i/d_model))

Even dimensions use sine; odd dimensions use cosine. This gives every position a unique fingerprint, and the relative position between two tokens can be expressed as a linear function of these encodings — allowing the model to generalise to unseen lengths.

The encoding is not learned — it is computed once, stored as a register_buffer (part of model state but not a learnable parameter), and added to every embedding at forward-pass time.

Python · PositionalEncoding 
import torch
import torch.nn as nn
import math

class PositionalEncoding(nn.Module):
    """Sinusoidal positional encoding added to token embeddings."""
    def __init__(self, d_model, dropout=0.1, max_len=5000):
        super().__init__()
        self.dropout = nn.Dropout(p=dropout)

        # pe: (max_len, d_model) — one encoding vector per position
        pe       = torch.zeros(max_len, d_model)
        position = torch.arange(0, max_len, dtype=torch.float).unsqueeze(1)  # (max_len, 1)

        # div_term = 1 / 10000^(2i/d_model)  — geometric frequency progression
        div_term = torch.exp(
            torch.arange(0, d_model, 2).float() * (-math.log(10000.0) / d_model)
        )

        pe[:, 0::2] = torch.sin(position * div_term)   # even dims ← sin
        pe[:, 1::2] = torch.cos(position * div_term)   # odd  dims ← cos

        # Shape: (max_len, 1, d_model) — broadcast over batch in forward()
        pe = pe.unsqueeze(0).transpose(0, 1)
        self.register_buffer('pe', pe)   # saved with model, not trainable

    def forward(self, x):
        # x: (seq_len, batch, d_model)
        x = x + self.pe[:x.size(0), :]   # broadcast: pe is (seq_len, 1, d_model)
        return self.dropout(x)
Positional Encoding — construction position = [0, 1, 2, …, max_len-1] shape (max_len, 1) div_term = [1/10000^(0/d), 1/10000^(2/d), …] shape (d_model/2,) pe[pos, 0] = sin(pos · div_term[0]) ← long wavelength (slow oscillation) pe[pos, 1] = cos(pos · div_term[0]) pe[pos, 2] = sin(pos · div_term[1]) ← shorter wavelength pe[pos, 3] = cos(pos · div_term[1]) ... pe[pos, d-2] = sin(pos · div_term[d/2-1]) ← very short wavelength pe[pos, d-1] = cos(pos · div_term[d/2-1]) register_buffer('pe', pe) ← saved in state_dict, moved with .to(device), not gradient tracked forward: x ← token_embedding(tokens) * √d_model shape (T, B, d_model) x ← x + pe[:T, :] broadcast over B x ← Dropout(x)
Why √d_model scaling?

The embedding weights are initialised with small values; without scaling, the positional encoding (values in [−1, 1]) would dominate the embedding signal. Multiplying embeddings by √d_model brings their typical L2 norm to the same order as the positional encoding vectors, letting the model balance both signals.

Transformer Encoder Layer

Multi-head self-attention + feed-forward + residual + LayerNorm

One encoder layer has two sub-layers, each wrapped with a residual connection and LayerNorm:

  • Multi-Head Self-Attention — every token attends to every other token in the source. Output size equals input size.
  • Position-wise FFN — two linear layers with a GELU activation. Expands to dim_feedforward (typically 4×d_model) then back to d_model.

The pre-norm vs post-norm convention matters. This implementation uses post-norm (original paper): residual first, then LayerNorm. Modern GPT-style models typically use pre-norm (LayerNorm before the sub-layer) for better training stability.

Python · TransformerEncoderLayer 
class TransformerEncoderLayer(nn.Module):
    def __init__(self, d_model, nhead, dim_feedforward=2048, dropout=0.1):
        super().__init__()
        # ── Sub-layer 1: Multi-Head Self-Attention ────────────────────
        # batch_first=False → inputs are (seq_len, batch, d_model)
        self.self_attn = nn.MultiheadAttention(
            d_model, nhead, dropout=dropout, batch_first=False
        )
        # ── Sub-layer 2: Position-wise FFN ───────────────────────────
        self.linear1    = nn.Linear(d_model, dim_feedforward)
        self.dropout    = nn.Dropout(dropout)
        self.linear2    = nn.Linear(dim_feedforward, d_model)
        self.activation = nn.GELU()
        # ── Layer Norms (post-norm style) ─────────────────────────────
        self.norm1 = nn.LayerNorm(d_model)
        self.norm2 = nn.LayerNorm(d_model)
        # ── Dropout on residual paths ─────────────────────────────────
        self.dropout1 = nn.Dropout(dropout)
        self.dropout2 = nn.Dropout(dropout)

    def forward(self, src, src_mask=None, src_key_padding_mask=None):
        # src: (T, B, d_model)
        # Sub-layer 1 — Self-Attention
        src2, attn_weights = self.self_attn(
            src, src, src,                        # Q=K=V=src (self-attention)
            attn_mask=src_mask,                   # optional additive mask
            key_padding_mask=src_key_padding_mask # mask padding tokens
        )
        src = self.norm1(src + self.dropout1(src2))  # residual + LayerNorm

        # Sub-layer 2 — FFN
        src2 = self.linear2(
            self.dropout(self.activation(self.linear1(src)))
        )
        src = self.norm2(src + self.dropout2(src2))  # residual + LayerNorm
        return src
Encoder Layer — Data flow (T = seq_len, B = batch, D = d_model) src (T, B, D) ├──────────────────────────────────────────┐ residual ↓ │ MultiheadAttention Q=K=V=src │ out (T, B, D) │ Dropout │ └─────────────────── + ────────────────────┘ ↓ LayerNorm (T, B, D) ├──────────────────────────────────────────┐ residual ↓ │ Linear(D → 4D) → GELU → Dropout → Linear(4D → D) Dropout │ └─────────────────── + ────────────────────┘ ↓ LayerNorm (T, B, D) ← output of encoder layer
04

Transformer Encoder

Stacking N identical encoder layers with optional final LayerNorm

Python · TransformerEncoder 
class TransformerEncoder(nn.Module):
    """Stacks num_layers copies of encoder_layer sequentially."""
    def __init__(self, encoder_layer, num_layers, norm=None):
        super().__init__()
        self.encoder_layer = encoder_layer   # single shared layer instance
        self.num_layers    = num_layers
        self.norm          = norm            # optional final LayerNorm

    def forward(self, src, mask=None, src_key_padding_mask=None):
        output = src
        for _ in range(self.num_layers):
            output = self.encoder_layer(
                output,
                src_mask=mask,
                src_key_padding_mask=src_key_padding_mask
            )
        if self.norm is not None:
            output = self.norm(output)
        return output   # (T_src, B, d_model)  — the "memory" for the decoder
Shared vs Independent Layers

This implementation reuses the same encoder_layer instance in the loop, meaning all layers share weights. The standard Transformer uses independent layers (one nn.ModuleList entry per layer, each with its own parameters). To get independent layers, use nn.ModuleList([copy.deepcopy(encoder_layer) for _ in range(num_layers)]) and iterate over it. Weight sharing reduces parameter count but also reduces model capacity.

05

Transformer Decoder Layer

Masked self-attention + cross-attention + FFN

The decoder layer adds a crucial third sub-layer — cross-attention — on top of the two sub-layers from the encoder:

  • Masked Multi-Head Self-Attention — the decoder attends to itself, but a causal mask blocks each position from seeing future positions. This enforces the autoregressive property: when generating token t, the decoder only sees tokens 1 … t−1.
  • Multi-Head Cross-Attention — queries come from the decoder, keys and values come from the encoder's memory. This is how the decoder reads the source sequence.
  • Position-wise FFN — identical to the encoder FFN.
Python · TransformerDecoderLayer 
class TransformerDecoderLayer(nn.Module):
    def __init__(self, d_model, nhead, dim_feedforward=2048, dropout=0.1):
        super().__init__()
        # ── Sub-layer 1: Masked Self-Attention ───────────────────────
        self.self_attn     = nn.MultiheadAttention(d_model, nhead, dropout=dropout, batch_first=False)
        # ── Sub-layer 2: Cross-Attention  (Q←tgt, K/V←memory) ───────
        self.multihead_attn = nn.MultiheadAttention(d_model, nhead, dropout=dropout, batch_first=False)
        # ── Sub-layer 3: FFN ─────────────────────────────────────────
        self.linear1    = nn.Linear(d_model, dim_feedforward)
        self.dropout    = nn.Dropout(dropout)
        self.linear2    = nn.Linear(dim_feedforward, d_model)
        self.activation = nn.GELU()
        # ── Three LayerNorms ─────────────────────────────────────────
        self.norm1 = nn.LayerNorm(d_model)   # after self-attn
        self.norm2 = nn.LayerNorm(d_model)   # after cross-attn
        self.norm3 = nn.LayerNorm(d_model)   # after FFN
        self.dropout1 = nn.Dropout(dropout)
        self.dropout2 = nn.Dropout(dropout)
        self.dropout3 = nn.Dropout(dropout)

    def forward(self, tgt, memory,
                tgt_mask=None, memory_mask=None,
                tgt_key_padding_mask=None, memory_key_padding_mask=None):
        # Sub-layer 1: Masked Self-Attention
        tgt2, _ = self.self_attn(
            tgt, tgt, tgt,                         # Q=K=V=tgt
            attn_mask=tgt_mask,                    # causal mask blocks future positions
            key_padding_mask=tgt_key_padding_mask
        )
        tgt = self.norm1(tgt + self.dropout1(tgt2))

        # Sub-layer 2: Cross-Attention
        tgt2, _ = self.multihead_attn(
            tgt, memory, memory,                   # Q=tgt, K=V=encoder memory
            attn_mask=memory_mask,
            key_padding_mask=memory_key_padding_mask
        )
        tgt = self.norm2(tgt + self.dropout2(tgt2))

        # Sub-layer 3: FFN
        tgt2 = self.linear2(self.dropout(self.activation(self.linear1(tgt))))
        tgt  = self.norm3(tgt + self.dropout3(tgt2))
        return tgt
Decoder Layer — Data flow tgt (T_tgt, B, D) memory (T_src, B, D) from encoder ├───────────────────────────────┐ ↓ │ residual Masked MultiheadAttention Q=K=V=tgt causal tgt_mask blocks future │ └──────────── + ────────────────┘ → LayerNorm ├───────────────────────────────┐ ↓ Q ← tgt │ residual Cross MultiheadAttention K=V=memory ←── memory decoder reads encoder context │ └──────────── + ────────────────┘ → LayerNorm ├───────────────────────────────┐ ↓ │ residual FFN Linear→GELU→Dropout→Linear │ └──────────── + ────────────────┘ → LayerNorm ↓ output (T_tgt, B, D)

Transformer Decoder

Stacking M decoder layers with shared memory threading through each

Python · TransformerDecoder 
class TransformerDecoder(nn.Module):
    def __init__(self, decoder_layer, num_layers, norm=None):
        super().__init__()
        self.decoder_layer = decoder_layer
        self.num_layers    = num_layers
        self.norm          = norm

    def forward(self, tgt, memory,
                tgt_mask=None, memory_mask=None,
                tgt_key_padding_mask=None, memory_key_padding_mask=None):
        output = tgt
        for _ in range(self.num_layers):
            output = self.decoder_layer(
                output, memory,                     # memory is same at every layer
                tgt_mask=tgt_mask,
                memory_mask=memory_mask,
                tgt_key_padding_mask=tgt_key_padding_mask,
                memory_key_padding_mask=memory_key_padding_mask
            )
        if self.norm is not None:
            output = self.norm(output)
        return output   # (T_tgt, B, d_model)

Full Encoder–Decoder Transformer

Assembling all components and adding Xavier weight initialisation

Python · Transformer (full model) 
class Transformer(nn.Module):
    def __init__(self, ntoken_src, ntoken_tgt, d_model, nhead,
                 dim_feedforward, num_encoder_layers, num_decoder_layers, dropout=0.1):
        super().__init__()
        self.d_model = d_model

        # ── Encoder side ──────────────────────────────────────────────
        self.encoder_embedding  = nn.Embedding(ntoken_src, d_model)
        self.pos_encoder        = PositionalEncoding(d_model, dropout)
        enc_layer               = TransformerEncoderLayer(d_model, nhead, dim_feedforward, dropout)
        self.transformer_encoder = TransformerEncoder(enc_layer, num_encoder_layers, nn.LayerNorm(d_model))

        # ── Decoder side ──────────────────────────────────────────────
        self.decoder_embedding  = nn.Embedding(ntoken_tgt, d_model)
        self.pos_decoder        = PositionalEncoding(d_model, dropout)
        dec_layer               = TransformerDecoderLayer(d_model, nhead, dim_feedforward, dropout)
        self.transformer_decoder = TransformerDecoder(dec_layer, num_decoder_layers, nn.LayerNorm(d_model))

        # ── Output projection: d_model → target vocab logits ──────────
        self.output_projection = nn.Linear(d_model, ntoken_tgt)
        self._init_weights()

    def _init_weights(self):
        # Xavier uniform initialises 2-D+ weight tensors to avoid
        # vanishing / exploding gradients at the start of training.
        for p in self.parameters():
            if p.dim() > 1:
                nn.init.xavier_uniform_(p)

    def _bool_to_additive_mask(self, mask, device, dtype):
        # Convert boolean padding mask (True=pad) to float additive mask (-inf=pad).
        if mask is None: return None
        if mask.dtype == torch.bool:
            return torch.where(mask,
                torch.tensor(float('-inf'), device=device, dtype=dtype),
                torch.tensor(0.0, device=device, dtype=dtype))
        return mask

    def forward(self, src, tgt,
                src_mask=None, tgt_mask=None,
                src_key_padding_mask=None, tgt_key_padding_mask=None,
                memory_key_padding_mask=None):
        # ── Encode source ─────────────────────────────────────────────
        src    = self.encoder_embedding(src) * math.sqrt(self.d_model)
        src    = self.pos_encoder(src)
        memory = self.transformer_encoder(
            src,
            mask=src_mask,
            src_key_padding_mask=self._bool_to_additive_mask(
                src_key_padding_mask, src.device, torch.float)
        )
        # ── Decode target ─────────────────────────────────────────────
        tgt    = self.decoder_embedding(tgt) * math.sqrt(self.d_model)
        tgt    = self.pos_decoder(tgt)
        output = self.transformer_decoder(
            tgt, memory,
            tgt_mask=tgt_mask,
            tgt_key_padding_mask=self._bool_to_additive_mask(
                tgt_key_padding_mask, tgt.device, torch.float),
            memory_key_padding_mask=self._bool_to_additive_mask(
                memory_key_padding_mask, memory.device, torch.float)
        )
        # ── Project to vocabulary logits ──────────────────────────────
        return self.output_projection(output)  # (T_tgt, B, ntoken_tgt)
⚖️

Xavier Uniform Init

Initialises every 2-D parameter with nn.init.xavier_uniform_. Sets the gain so that the variance of activations and gradients is preserved through each layer, preventing explosion or vanishing at initialisation.

  • Formula: U(−a, a) where a = gain × √(6 / (fan_in + fan_out))
  • Assumes linear activation (not exact for GELU, but still effective)
  • Skips 1-D tensors (biases, LayerNorm weight/bias)
  • Alternative: Kaiming (He) init is preferred for ReLU-heavy networks
🔀

Boolean → Additive Mask Conversion

PyTorch's nn.MultiheadAttention accepts either a boolean mask or a float additive mask. When mixing both (e.g., one boolean padding mask and no attention mask), it emits a deprecation warning. The _bool_to_additive_mask helper converts True → -inf, False → 0.0 to keep all masks as floats.

  • Attention weight for -inf position → 0 after softmax
  • 0.0 position → unaffected (multiplied by exp(0) = 1)
  • PyTorch 2.1+: use is_causal=True for causal mask (avoids materialising matrix)

Masking Utilities

Causal mask and padding mask — what they look like and why they are needed

Python · generate_square_subsequent_mask & create_padding_mask 
def generate_square_subsequent_mask(size):
    """Upper-triangular -inf mask — prevents attending to future tokens."""
    # Step 1: lower-triangular boolean matrix (True where attention IS allowed)
    mask = (torch.triu(torch.ones(size, size)) == 1).transpose(0, 1)
    # Step 2: convert: allowed → 0.0,  blocked → -inf
    mask = mask.float()\
               .masked_fill(mask == 0, float('-inf'))\
               .masked_fill(mask == 1, float(0.0))
    return mask   # shape (size, size)


def create_padding_mask(seq, pad_idx):
    """Boolean mask: True where the token is a padding token."""
    # seq: (seq_len, batch) with batch_first=False convention
    # MultiheadAttention expects key_padding_mask: (batch, seq_len)
    if seq.dim() == 2 and seq.size(0) > seq.size(1):
        padding_mask = (seq == pad_idx).transpose(0, 1)  # → (batch, seq_len)
    else:
        padding_mask = (seq == pad_idx)
    return padding_mask   # dtype=bool

Mask shapes summary

tgt_mask (causal): (T_tgt, T_tgt) — square, one row per query position. Added to attention logits before softmax. key_padding_mask: (batch, seq_len) — one row per sequence in the batch. True means "ignore this key". When both are used, convert both to float additive form to avoid the PyTorch deprecation warning.

Next Steps

Training loop, inference, and production upgrades

Training Loop

Implement teacher forcing: feed ground-truth shifted-right tokens as tgt during training. Compute CrossEntropyLoss on logits.view(-1, vocab_size) against labels.view(-1). Use AdamW with cosine LR schedule and gradient clipping at 1.0.

Autoregressive Inference

At inference time, start tgt = [<bos>] and loop: run forward pass, sample/argmax the last position's logit, append the predicted token to tgt, repeat until <eos> or max length. Use KV-cache to avoid recomputing past keys/values.

Replace Shared Layer with ModuleList

The current encoder/decoder reuses one layer instance for all N layers, meaning all layers share the same weights. For full expressivity, replace with nn.ModuleList([copy.deepcopy(layer) for _ in range(N)]) so each layer has independent parameters.

Switch to Pre-Norm

Modern transformers (GPT, LLaMA) apply LayerNorm before each sub-layer (pre-norm) rather than after (post-norm). Pre-norm improves training stability for deep models without requiring careful learning-rate warmup. Change norm(x + sublayer(x)) to x + sublayer(norm(x)).

RoPE Instead of Sinusoidal PE

Rotary Position Embedding (RoPE) encodes relative positions by rotating query and key vectors before computing attention. It generalises better to unseen lengths and is used by LLaMA, Mistral, and most modern LLMs. See Chapter 5 → Positional Encoding for the full RoPE implementation.

Flash Attention

Replace nn.MultiheadAttention with F.scaled_dot_product_attention(..., is_causal=True) (PyTorch 2.0+). This dispatches to Flash Attention automatically, reducing attention memory from O(N²) to O(N) and speeding up training by 2–4×.
