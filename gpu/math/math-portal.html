Key Formulas at a Glance
The most important equations — one per concept
Dot Product (Ch 1)
a · b = Σᵢ aᵢbᵢ = ‖a‖ ‖b‖ cos(θ)
Attention score = q · k / √d_k
RMS Norm (Ch 1)
‖x‖_rms = √(mean(x²)) x_norm = x / ‖x‖_rms
Used in LLaMA, GPT-NeoX instead of LayerNorm
Matrix Multiply (Ch 2)
C[i,j] = Σₖ A[i,k] · B[k,j] (m×k) @ (k×n) → (m×n)
Every linear layer is y = x @ W
SVD / LoRA (Ch 3)
M = U Σ Vᵀ ΔW = B × A (r ≪ d)
LoRA: 48× fewer params at rank r=8
Chain Rule (Ch 4)
P(w₁..wₙ) = Πᵢ P(wᵢ | w₁..wᵢ₋₁)
The math behind autoregressive LLMs
Softmax Temperature (Ch 5)
P(i) = exp(lᵢ/τ) / Σⱼ exp(lⱼ/τ)
τ→0 = greedy, τ=1 = default, τ>1 = creative
Cross-Entropy Loss (Ch 6)
L = -(1/N) Σᵢ log P(wᵢ|ctx) PPL = exp(L)
MLE objective = NLL = cross-entropy
KL Divergence (Ch 6)
DKL(p‖q) = Σ p log(p/q) ≥ 0, = 0 iff p = q
RLHF penalty: L = reward − β·DKL
AdamW (Ch 8)
m ← β₁m + (1−β₁)g v ← β₂v + (1−β₂)g² θ ← θ(1−ηλ) − η·m̂/√v̂
β₁=0.9, β₂=0.95, λ=0.1 for LLMs
Scaled Attention (Ch 9)
Attn = softmax(QKᵀ/√d_k)·V score std ≈ 1 when scaled
√d_k prevents softmax saturation
KV Cache Memory (Ch 10)
bytes/token = 2 × L × H_kv × d_h × dtype
LLaMA-3 8B: ~0.125 MB per token
BW Bound (Ch 10)
tok/s ≤ memory_bw / model_bytes
H100: 3.35 TB/s ÷ 140 GB ≈ 24 tok/s
