Introduction to GPUs

A Graphics Processing Unit (GPU) is a specialized processor designed to accelerate the rendering of images, animations, and video. Unlike CPUs, which handle general-purpose computing, GPUs excel at performing many simple calculations simultaneously — a capability known as parallel processing.

Fun Fact: The first commercial GPU, the NVIDIA GeForce 256 (1999), introduced hardware transform and lighting (T&L), offloading these tasks from the CPU.

History & Evolution

1970s–80s: Early Graphics

Fixed-function hardware in arcade machines and early PCs (e.g., IBM CGA, EGA).

1990s: 3D Acceleration

3dfx Voodoo, NVIDIA RIVA 128, ATI Rage — birth of consumer 3D graphics cards.

2000s: Programmable Shaders

DirectX 8/9, OpenGL — introduction of vertex and pixel shaders.

2010s–Now: GPGPU & AI

CUDA (2006), Tensor Cores, RT Cores, DLSS, and AI acceleration.

GPU Architecture

Modern GPUs use SIMD (Single Instruction, Multiple Data) architecture with thousands of small cores grouped into:

  • Streaming Multiprocessors (SMs) – Execution units in NVIDIA GPUs
  • Compute Units (CUs) – AMD’s equivalent
  • Tensor Cores – Specialized for matrix math in AI
  • Ray Tracing (RT) Cores – Accelerate BVH traversal and intersection

CUDA & Parallel Computing

NVIDIA’s CUDA (Compute Unified Device Architecture) allows developers to use GPUs for general-purpose computing (GPGPU).

// Simple CUDA Kernel
__global__ void add(int *a, int *b, int *c, int n) {
    int idx = blockIdx.x * blockDim.x + threadIdx.x;
    if (idx < n) c[idx] = a[idx] + b[idx];
}

Threads → Blocks → Grid hierarchy enables massive parallelism.

AMD ROCm & RDNA

AMD competes with RDNA (gaming) and CDNA (compute) architectures.

  • ROCm – Open-source platform for GPU computing
  • Infinity Cache – Large on-die cache reduces memory latency
  • FSR 3 – FidelityFX Super Resolution with frame generation

GPUs in AI & Machine Learning

Gaming & Ray Tracing

Modern games use:

  • Real-time Ray Tracing – Physically accurate lighting
  • DLSS / FSR / XeSS – AI upscaling for higher FPS
  • Variable Rate Shading (VRS) – Optimize performance

Understanding GPU Specs

Spec Meaning

Boost Clock Max frequency under load
VRAM Dedicated video memory (GDDR6X, HBM3)
Tensor Cores AI acceleration units
