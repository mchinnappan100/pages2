What You'll Learn

Core skills and knowledge gained across the complete series

🏗️

Transformer Internals

How attention, RoPE, SwiGLU FFN, RMSNorm, and residual connections combine to form a modern LLM — with complete PyTorch implementations.

🎯

Alignment Pipeline

The three-stage RLHF pipeline, Bradley-Terry reward models, PPO objectives, and the simpler DPO formulation that replaced PPO in most production pipelines.

🔬

Efficient Fine-Tuning

LoRA's low-rank decomposition, QLoRA's 4-bit base with BF16 adapters, and how to fine-tune a 70B model on a single A100 in under 24 hours.

🗜️

Quantization Methods

From absmax INT8 to GPTQ second-order quantization and AWQ activation-aware scaling — plus when to use each based on the quality-memory tradeoff.

💾

KV Cache Economics

The exact memory formula, why long contexts are so expensive, and how PagedAttention, GQA, and FP8 KV cache cut memory waste to near-zero.

⚡

Flash Attention 2

Why standard attention is O(N²) in memory, how SRAM tiling achieves O(N), and how to enable it in one line with PyTorch 2.0+.

🐍

PyTorch Inference

From a minimal greedy loop to production inference with KV cache, sampling strategies, speculative decoding, and quantized model serving.

🔢

JAX & Flax Inference

JIT-compiled autoregressive generation, explicit PRNG keys, vmap for vectorized batch processing, and pmap for multi-device tensor parallelism.

🏭

Production Serving

vLLM with continuous batching and PagedAttention, TensorRT-LLM, llama.cpp for local deployment, and real throughput benchmarks across hardware.
