Google CCaaS Integration

Google Cloud Contact Center AI (CCaaS)

Google’s Contact Center as a Service (CCaaS) — known as Contact Center AI Platform (CCAI Platform) — provides omnichannel routing, virtual agents, agent assist, CRM integrations, and powerful analytics.

Architecture Diagram

Customer (Phone) ──────► Google CCAI Telephony Gateway ──────► Dialogflow CX ──────► Webhooks ──────► Backend APIs
                                 │                                │
Customer (Chat Widget) ──────────┘                                └────► Agent Assist / Analytics
                                  
Agent Desktop ◄────────────────────────── Agent Assist API ◄──────── Knowledge Base / CRM

This architecture supports both voice and chat routing, integrates with backend systems, and provides real-time support for human agents.

Webhooks & API Integration

Webhook Request Example

{
  "fulfillmentInfo": { "tag": "getAccountStatus" },
  "sessionInfo": {
    "parameters": { "accountId": "12345" }
  }
}

Webhook Response Example

{
  "fulfillment_response": {
    "messages": [
      { "text": { "text": ["Your account balance is $42.80."] } }
    ]
  }
}

Salesforce Integration (Step-by-Step)

  1. Create or configure your Dialogflow CX Agent.
  2. Enable CCAI APIs: Dialogflow CX, Agent Assist, Contact Center AI.
  3. Install Google Contact Center AI package in Salesforce AppExchange.
  4. Connect Salesforce Omni-Channel with Dialogflow routing.
  5. Configure:
    • Voice or Chat routing
    • Agent Assist (real-time suggestions)
    • Knowledge Article sync
  6. Set up Salesforce → CCAI authentication (OAuth service account).
  7. Add “CCAI Agent Assist” panel to Salesforce console apps.
  8. Test interactions:
    • Chat flows
    • Voice flows
    • Case creation

Salesforce receives conversation transcripts, agent assist summaries, and recommended actions.
