‚öôÔ∏è Setup & Installation

üìã Prerequisites

  • Salesforce org with Lightning Web Components enabled
  • Apex controller: CsvExportController
  • Access to Salesforce matrix report JSON exports
  • ContentVersion object create permissions for CSV export

Step 1: Deploy Apex Controller

Create the CsvExportController Apex class:

CsvExportController.cls

public with sharing class CsvExportController {
    @AuraEnabled
    public static String generateCSVFile(String csvData, String fileName) {
        try {
            // Decode Base64 CSV data
            Blob csvBlob = EncodingUtil.base64Decode(csvData);
            
            // Create ContentVersion record
            ContentVersion cv = new ContentVersion();
            cv.Title = fileName;
            cv.PathOnClient = fileName;
            cv.VersionData = csvBlob;
            cv.FirstPublishLocationId = UserInfo.getUserId();
            insert cv;
            
            // Retrieve ContentDocumentId for download URL
            cv = [SELECT ContentDocumentId 
                  FROM ContentVersion 
                  WHERE Id = :cv.Id 
                  LIMIT 1];
            
            // Return download URL
            return '/sfc/servlet.shepherd/document/download/' + cv.ContentDocumentId;
            
        } catch (Exception e) {
            throw new AuraHandledException('Error generating CSV: ' + e.getMessage());
        }
    }
}

üîë Understanding FactMap Keys

Key Pattern Description Example
T!T Grand Total Overall aggregates for entire report
0_T Row Group Total Total for first row grouping
T_0 Column Group Total Total for first column grouping
0_0 Cell Intersection Value at row 0, column 0

Component Properties

Computed Properties (Getters)

totalPages

Calculates total number of pages based on records and page size.

get totalPages() {
    return Math.ceil(this.totalRecords / this.pageSize);
}

isPreviousDisabled

Returns true when on first page, disabling "Previous" button.

get isPreviousDisabled() {
    return this.currentPage === 1;
}

isNextDisabled

Returns true when on last page, disabling "Next" button.

get isNextDisabled() {
    return this.currentPage * this.pageSize >= this.totalRecords;
}

Component Methods

handleFileUpload(event)

Processes uploaded JSON file and initiates data parsing.

Parameters: event (File input change event)

Returns: void

closeBalloon()

Hides the JSON preview balloon.

Parameters: None

Returns: void

processMatrixReport(data)

Parses factMap structure and generates table data.

Parameters: data (Object) - Parsed JSON object

Returns: void

updatePagedData()

Updates the pagedMatrixData based on current page and page size.

Parameters: None

Returns: void

handlePageChange(event)

Handles pagination button clicks (Previous/Next).

Parameters: event (Button click event with data-direction)

Returns: void

handleSort(event)

Sorts the matrix data by selected column and direction.

Parameters: event (Datatable sort event)

Returns: void

generateCSV()

Converts matrix data to Base64-encoded CSV string.

Parameters: None

Returns: String (Base64 CSV) or null

handleExportCSV()

Calls Apex to generate downloadable CSV file.

Parameters: None

Returns: void (Shows toast notification)

showToast(title, message, variant)

Displays Lightning toast notification.

Parameters: title (String), message (String), variant (String: 'success'|'error'|'warning'|'info')

Returns: void

üîç Troubleshooting

‚ùå No Data Displayed After Upload

Possible Causes & Solutions:

  • Verify JSON file contains "factMap" property
  • Check that factMap contains aggregate data
  • Ensure JSON is valid (use online validator)
  • Review browser console for parsing errors

‚ùå CSV Export Failing

Possible Causes & Solutions:

  • Confirm user has create permission on ContentVersion
  • Check CsvExportController is deployed correctly
  • Verify Apex method is @AuraEnabled
  • Review debug logs for Apex errors
  • Ensure data exists before export attempt

‚ùå Pagination Not Working

Possible Causes & Solutions:

  • Check that totalRecords is being set correctly
  • Verify pageSize is a valid positive number
  • Ensure updatePagedData() is called after data changes
  • Review event handler bindings on buttons

‚ùå Columns Not Displaying Correctly

Possible Causes & Solutions:

  • Verify aggregates array structure in factMap
  • Check that first row has aggregates defined
  • Ensure column generation loop is executing
  • Review matrixData structure in console

‚ùå Sorting Not Working

Possible Causes & Solutions:

  • Confirm columns have sortable: true
  • Check handleSort event is properly bound
  • Verify fieldName matches column definitions
  • Ensure data values are comparable (strings/numbers)

‚ú® Best Practices

‚úÖ Do's

  • Export matrix reports in JSON format from Salesforce
  • Validate JSON structure before uploading
  • Test with small datasets first
  • Use descriptive report names in JSON
  • Keep factMap keys organized and logical
  • Monitor page size for performance
  • Handle error cases gracefully

‚ùå Don'ts

  • Don't upload extremely large JSON files (>10MB)
  • Don't modify JSON structure manually without validation
  • Don't forget to test CSV export permissions
  • Don't ignore console errors during development
  • Don't skip error handling in Apex controller
  • Don't hardcode values - use dynamic processing

‚ö° Performance Optimization

üìä Data Volume

For reports with thousands of records, consider:

  • Keeping page size at 10-20 records
  • Pre-filtering data in Salesforce before export
  • Using browser with good JSON parsing performance

üîÑ Rendering Optimization

Improve render performance by:

  • Using pagination to limit DOM elements
  • Avoiding re-rendering entire dataset on sort
  • Implementing virtual scrolling for very large datasets

üíæ Memory Management

Manage memory efficiently:

  • Close balloon after reviewing JSON to free memory
  • Clear previous data when uploading new file
  • Monitor browser memory usage with large files
