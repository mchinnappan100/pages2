🎼 ABC Notation Guide

Learn how to write music using ABC notation.

What is ABC Notation?

ABC notation is a simple text-based music notation format for writing and sharing music. Instead of graphical notation, musical elements such as notes, measures, tempo, and key are written using plain text.

ABC notation is widely used in folk music communities and supported by software such as music renderers and editors.

Basic Structure

Every ABC tune starts with a header followed by the musical notation. 

X:1
T:My Tune
M:4/4
K:C
C D E F | G A B c |

Important Header Fields

Field Meaning Example
X: Reference Number X:1
T: Title T:Amazing Tune
M: Meter M:4/4
K: Key Signature K:G

Writing Notes

Notes are written using letters A–G.

Example

K:C
C D E F | G A B c |

Rests

The letter z represents a rest. 

K:C
C D z E | F G z2 |

Measures (Bars)

Bars divide music into measures using the | symbol. 

C D E F | G A B c |

Note Durations

Notation Meaning
C Default duration
C2 Double length
C/2 Half length

Example Tune 

X:1
T:Twinkle Twinkle
M:4/4
K:C
C C G G | A A G2 |
F F E E | D D C2 |
