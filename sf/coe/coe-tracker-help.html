Getting Started

1. Create a Project

  • Click + New Project in the header
  • Give it a title and optional description
  • Click Create

2. Add Activities

  • Inside a project, click + Add Activity
  • Fill in title, category, status, priority, assignee, due date, notes
  • Choose recurrence if needed (daily/weekly/every 15 days/monthly)
  • Click Add Activity or press Enter

3. Manage & Track

  • Drag & drop to reorder activities
  • Click pencil icon to edit
  • Mark as Completed when done
  • Use filters (All, Open, Done, Blocked, High Priority)
